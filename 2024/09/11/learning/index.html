课程学习

大学的课程成绩组成往往有平时分和期末考试或者大作业等组成，不再像高中那样子单一

硬课

种类

  • 通识必修：微积分、线代、大物、（化学）、外语
  • 专业基础课程&专业课：详见自己专业的培养方案

学习方法

  • 注重构建知识体系：理解课程之间的联系，形成系统化的知识结构。
  • 明确课程培养目标和考核要求：了解课程的培养目标和考核方式，有针对性地学习。
  • 选择适合自己的学习方式：找到最适合自己的学习方式，无论是预习-上课-复习-练习的传统模式，还是自学-上课补充-复习-练习的自主模式，也可以是bilibili大学+智云课堂/mooc。
  • 即时复习，不建议考试周才补天，避免临时抱佛脚，定期复习巩固知识。

水课

种类

  • 思政课
  • 通识选修（含通识核心）

学习方法

  • 不要抱有卷的心态去学（当然也行
  • 不需要过于注重绩点（当然也行
  • 明确课程考核要求
  • 思政课尽可能抓住可以加分的机会，比如当组长，多发言
  • 尽可能提前完成论文、读书报告等
  • 小组作业&pre相关详见后文

复习模式

以下是几点复习建议

  • 合理利用历年卷，通过历年试卷了解考试重点，有针对性地复习（面向历年卷学习
  • 合理安排论文、读书报告等大作业的时间，以免忘记或遗漏
  • 复（yu）习真的很重要

时间规划

规划大学时间是一个重要的技能，可以帮助你更有效地学习、发展个人兴趣、参与社交活动以及准备未来的职业生涯。

  • 制定计划：为每个学期、每个月、甚至每天制定详细的学习计划。
  • 优先级排序：根据任务的紧急性和重要性，合理安排时间。

月计划

  1. 日历
    • 用来记录每个月的DDLs、重要节日
    • 安排一些长期项目的子任务（如期末复习）
    • 也可以将课表导入日历，当然也可以使用第三方工具查看课表
  2. 每月目标
    从学习、生活、开销等方面制定目标

*安利三个挺好用的工具：

  • celechron：手机app，可以看课表，做规划，查成绩等，朵朵搜一下或者去官网看看celechron.top
  • zju-learning-assistant:电脑程序，可以下载学长浙大的课件，可以看成绩、智云，98搜一下
  • 桌面日历，顾名思义

当然，也可以是桌面助手、电脑管家……

周计划&日计划

  1. 参照月计划标注DDL和重要时间点
    • 做计划的过程中，不仅仅是罗列每天的任务，更重要的是在写下任务的同时在脑子里把对应的任务安排到对应的时间块
  2. 每天待办事项没有做完的摘录到周计划中
  3. 如何确定自己的DDL
    • 你要想“当我最晚什么时候开始做这件事时，才不会影响完成的质量”
  4. 不要为了做计划而做计划，给自己留余地，每天留有一些时间给自己
  5. 计划的灵活选择和调整

生涯规划

大一大事记

在大一期间，多接触一些，不断明晰与调整自身的生涯规划，大一期间大概会经历怎么几件大事

  1. 第一轮类内专业确认
    1. 仔细阅读学校文件
    2. 多了解自己想学或感兴趣的专业
    3. 多和对应学院老师交流
    4. 热门专业做好面试/笔试准备
  2. 第二轮跨类专业确认
    1. 跨类的话多了解意向专业
    2. 准备好面试的心理准备
  3. 秋冬转专业
    1. 注意转专业要求，好像是专业确认的时候一起发，但是选课前需要注意（有点奇怪），要转专业的最好提前关注
    2. 卷绩点（？）各个专业的转专业容量有点少其实，高绩点能让你不会被表刷
  4. 春夏辅修
    1. 辅修可以提高你的低定底线
    2. ACEE（工高班）也是一种选择
  5. 大二评奖评优
    1. 打开学生手册评奖评优相关章节，奖学金政策我就不多赘述啦
    2. 主要的条件：
      • 思政、体测、综素……
      • 评定标兵，不同的要求不同，但是学业优秀标兵应该是评奖评优必要的
      • 加分（？班团委、社会工作、志愿者小时数、竞赛、其他荣誉
    3. 刚开学那阵子，园网、院网公示评奖评优的评定细则和成绩排名等，有需要的可以多多关注
    4. 当然，也可以关注一下学工部门户网，学工部会先发布通知，之后学院会按照通知执行任务并进行二次通知
    5. 之后申请标兵、奖学金各种荣誉，标兵应该是eta申请，奖学金申请之后要进行筛选，应该是要面试之类的，筛选之后填申请表并上交，学院上交学工部，归档/进一步评定

小组作业指北

开学季又到了，同学们的脸上洋溢着对新学期的期待和对假期的不舍，而随之而来的还有小组作业！尤其是思政课和通识核心课程（）

小组作业，它通常以小组PPT展示的形式出现，也就是我们常说的“pre”，全称presentation。这种作业形式在大学很常见，从思政课到通识课再到部分专业课，无论你是大一的小鲜肉还是大五的老油条，都逃不过它的“魔爪”。当然，小组作业不只有PPT展示，还有其他小组合作，例如现在思政课逐渐引入了“剧情演绎”，又如电工训的合作（，但其本质上都是与人合作完成作业。

在未来的日子里，你可能会和你的队友们一起熬夜赶工，一起在讲台上展示你们的成果，也可能会遇到那些让人哭笑不得的团队内部小插曲（比如谁当组长谁怨种。但不管怎样，这都是大学生活的一部分，是成长的一部分，也可能是回忆的一部分。

痛点

1.小组一生一起走，谁当组长谁怨种

  • 问题：组长通常需要协调团队成员、分配任务、确保项目进度，还可能遇到不积极甚至不配合的组员，这些都会导致组长感到压力较大，很多时候组长就是投骰子点数最小或者最大决定出来的大怨种。
  • 应对策略：emmmm难说，遇上了不配合的组员，也只能硬着头皮干完甚至超额完成任务，主要是对自己负责吧
    2.拖延严重，人均ddl战神
  • 问题：团队成员可能会拖延任务，直到最后期限（deadline）临近才开始行动，这可能会影响作业的质量和团队的士气。
  • 应对策略：提前准备好主题和设定好目标，鼓励团队成员提前开始工作，把ddl提前，多提醒组员完成该完成的任务。
    3.分工不当，互相甩锅
  • 问题：如果任务分配不均衡，一些成员可能会感到负担过重，而其他成员则可能参与不足。
  • 应对策略：组长在开始时征求组员意见，搭建好展示的框架，尽可能公平地分配任务，之后让组员自选想要完成的部分。

最佳解决方式

选不需要展示的或者只需要个人展示的课（bushi）

Pre步骤

  1. 确认主题：
    1. 确定你的展示主题，确保它既符合课程要求，也符合你的目标听众的兴趣，
    2. 当然可能是给定了主题，也可能是老师提供了多个可供选择的主题，还可能是自由选题
    3. 不建议选过于简单或者大众化的主题，老师可以会觉得过于老套而给分不高，除非你能从一个新颖的角度展开，让人眼前一亮。
  2. 搭建框架/大纲：
    1. 一般由组长完成
    2. 制定一个清晰的演讲结构，通常包括引言、主体（几个关键点）、结论
    3. 确定每个部分的主要信息和支持细节
  3. 查找资料
    1. 收集支持你论点的数据、案例研究、引用和证据
    2. 确保所有信息都是准确和可靠的
    3. 必要时可以在PPT结尾处放上你们的参考文献或资料
  4. PPT制作
    1. 详略得当，注重排版，突出重点，正文字号也别太小
    2. 保持幻灯片简洁，避免过多的文字，适当使用图表、图片和关键词来强化你的信息
    3. 不建议使用花里胡哨的PPT来展示，这可能不会是加分项，甚至会是扣分项
    4. 关于如何制作，参见后文
  5. 撰写讲稿
    1. 根据你的演讲大纲撰写讲稿，但不要逐字逐句地写
    2. 练习演讲，尽可能能够流畅自然地表达你的观点
  6. 做Pre
    1. 考虑使用计时器来确保你不会超时
    2. 确保所有准备工作都已经做到位
    3. 在Pre过程中尽可能脱稿，不要一直低头念稿或者单纯地念PPT，适当做一些互动或者交流性动作
    4. 尽可能呈现出自信大方的模样，

PPT制作方式

  1. 根据搜集的资料写好大纲内容，然后直接大纲生成PPT，然后自己适当排版一下
  2. 根据主题，自己一点点手搓，不是很建议这么做，因为这将是一项工程量巨大的工作
  3. 套用适合的PPT模板，填充需要的内容，如果小组作业的是每人做一部分，最好是协商好使用同一个PPT模板，否则可能出现不协调的情况
  4. AI生成/辅助，现在AI已经足够发达，生成个PPT还是不难的，甚至生成的比有些组员做的还好

小结

确实，团队合作不仅能锻炼个人的沟通和协作能力，还能在共同克服困难的过程中培养深厚的友谊。面对挑战时，保持积极的态度，相信团队的力量，往往能够带来意想不到的成果。

大学中很多友谊也是在团队协作收获的，在共同完成任务的过程中，我们往往能够建立起一种特殊的联系，这种联系有时甚至可以超越普通的友谊，因为它是在共同面对挑战和解决问题的过程中形成的。

就写这么多吧，祝愿各位能在小组作业中茁壮成长！

社会工作

这一节因为涉及到综素、二三课堂、竞赛等，所以我放在学习章节里

班团委

  • 班级建设：通过参与班级管理，能够得到一定的锻炼，也能增强团队精神和班级凝聚力
  • 信息获取：班团委能够更快了解学校和班级的最新动态和通知
  • 组织能力培养：通过组织班级活动，提升自己的领导力和协调能力
  • 功利的加分：参与班团委工作可能会为个人简历增色，有时还能获得额外的加分

学生组织/社团

  • 积累面试经验
  • 扩大交友圈，通过组织或者社团，结识不同专业的人，拓宽社交网络；加入社团，你们可能拥有共同的兴趣爱好，有更多的共同话题
  • 学生组织/社团可能会有丰富的团建活动，这能增进彼此的友谊，让整体更加团结、温馨
  • 参加组织/社团的各项活动，能够学习多项技能，也能培养一些兴趣爱好
  • *打比赛：有些组织可能就是为打比赛而生，所以感兴趣的也可以加入对应的组织
  • 另外，在学生组织/社团干满一年，获得工作证明，可以获得的1.5二课分

志愿活动

  • 实现自我价值：通过帮助他人和社会，获得成就感和满足感
  • 信息检索：在寻找和参与志愿活动的过程中，可能需要信息搜集和筛选能力
  • 评选志愿服务标兵
  • 评定星级志愿者
  • 三星级及以上志愿者可以申请第三课堂学分
  • 有些学院保研对志愿者小时数有所要求，更多的志愿者小时数也可能会有加分

How to balance?

  • 社会工作不贪多，适合自己更重要，选择与个人兴趣和职业规划相符合的社会工作，避免盲目追求数量
  • 学习之余，可以多做一些志愿活动之类的
  • 利弊的话，自己权衡吧
  • 更希望大家能有自己的想法，独立思考，做出最适合自己的选择。
