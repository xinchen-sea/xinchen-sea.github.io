课程学习

大学的课程成绩组成往往有平时分和期末考试或者大作业等组成，不再像高中那样子单一

硬课

种类

  • 通识必修：微积分、线代、大物、（化学）、外语
  • 专业基础课程&专业课

学习方法

  • 注重构建知识体系
  • 明确课程培养目标和考核要求
  • 选择适合自己的学习方式
  • 即时复习，不建议考试周才补天

水课

种类

  • 思政课
  • 通识选修

学习方法

  • 不要抱有卷的心态去学（当然也行
  • 不需要过于注重绩点（当然也行
  • 明确课程考核要求
  • 思政课尽可能抓住可以加分的机会
  • 尽可能提前完成论文、读书报告等
  • 小组作业&pre相关详见后文

学习模式

我觉得有以下这么几种学习方式，可以选择适合自己的

  • 方式1：预习-上课-复习-练习
  • 方式2：自学-上课补充-复习-练习
  • 方式3：bilibili-university/智云课堂
  • 方式4：……

复习模式

以下是几点复习建议

  • 合理利用历年卷，面向历年卷学习
  • 合理安排论文时间
  • 复（yu）习真的很重要

生涯规划

大一大事记

在大一期间，多接触一些，不断明晰与调整自身的生涯规划，大一期间大概会经历怎么几件大事

  1. 第一轮类内专业确认
    1. 仔细阅读学校文件
    2. 多了解自己想学或感兴趣的专业
    3. 多和对应学院老师交流
    4. 热门专业做好面试/笔试准备
  2. 第二轮跨类专业确认
    1. 跨类的话多了解意向专业
    2. 准备好面试的心理准备
  3. 秋冬转专业
    1. 注意转专业要求，好像是专业确认的时候一起发，但是选课前需要注意（有点奇怪），要转专业的最好提前关注
    2. 卷绩点（？）各个专业的转专业容量有点少其实，高绩点能让你不会被表刷
  4. 春夏辅修
    1. 辅修可以提高你的低定底线
    2. ACEE（工高班）也是一种选择
  5. 大二评奖评优
    1. 打开学生手册评奖评优相关章节，都写得很清楚，奖学金政策我不多赘述
    2. 主要的条件：
      • 思政、体测、综素……
      • 评定标兵，不同的要求不同，但是学业优秀标兵应该是评奖评优必要的
      • 加分（？班团委、社会工作、志愿者小时数、竞赛、其他荣誉
    3. 刚开学那阵子，园网、院网公示评奖评优的评定细则和成绩排名等，有需要的可以多多关注
    4. 当然，也可以关注一下学工部门户网，学工部会先发布通知，之后学院会按照通知执行任务并进行二次通知
    5. 之后申请标兵、奖学金各种荣誉，标兵应该是eta申请，奖学金申请之后要进行筛选，筛选之后填申请表并上交，学院上交学工部，归档/进一步评定

大一之后

至于大一之后emmm，我写下这些的时候才大二，提供不了太多信息，我觉得可能有以下这些

  1. 每年评奖评优
  2. 外设奖学金，不同学院有的奖学金不同
  3. 保研、考研、出国留学
  4. 对外交流

如果你想卷保研的话，建议不只在学习一个赛道卷，可以参考往年推免政策，做一些加分的项目……

社会工作

这一节因为涉及到综素、二三课堂、竞赛等，所以我放在学习章节里

班团委

  • 班级建设
  • 信息获取
  • 组织能力培养
  • 功利的加分

学生组织/社团

  • 积累面试经验
  • 扩大交友圈
  • 丰富的团建活动
  • 学习多项技能
  • *打比赛
  • 兴趣爱好
  • 第二课堂

志愿活动

  • 实现自我价值
  • 信息检索
  • 志愿标兵
  • 星级志愿者评定
  • 第三课堂
  • 保研加分

How to balance?

  • 社会工作不贪多，适合自己更重要
  • 学习之余，可以多做一些志愿活动之类的
  • 利弊的话，自己权衡吧
  • 更希望大家能有自己的想法
